# AI Review Bot Skill

This skill enables an agent to monitor GitHub Pull Requests, perform deep architectural reviews using Senso AI, and identify AI-generated "slop" through advanced heuristics.

## Usage

Use this skill when:
- You need to automate quality gates for incoming Pull Requests.
- You want to detect if code was generated by an LLM without human oversight.
- You need to identify complex architectural patterns or memory leaks.

## Requirements

1. **GitHub App**: Must have `Pull Request: Read & Write` permissions.
2. **Senso API**: A valid API key from [senso.ai](https://senso.ai).
3. **Environment**:
   - `GITHUB_APP_ID`
   - `GITHUB_PRIVATE_KEY_PATH`
   - `GITHUB_WEBHOOK_SECRET`
   - `SENSO_API_KEY`

## Workflow

1. **Deploy**: Run the `server.py` to start the webhook listener and dashboard.
2. **Configure**: Set the GitHub App webhook URL to your server's `/webhooks` endpoint.
3. **Monitor**: Access the dashboard at `http://localhost:3000` to view PR stats and manually block submissions.

## Tooling

This skill utilizes:
- **Railtracks**: For agentic pipeline orchestration.
- **Senso AI**: For knowledge-base driven code review.
- **Flask**: For the administrative interface.
